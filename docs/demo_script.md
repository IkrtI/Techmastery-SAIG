# Demo Video Script (3-4 นาที)

เตรียมก่อนอัด: backend + frontend รันอยู่, ประมวลผลวิดีโอครบ, สร้างรายงาน AI ไว้แล้ว 2-3 เหตุการณ์
(เหลือ 1 เหตุการณ์ไว้กดสดในคลิปเพื่อโชว์ว่า generate จริง)

| เวลา | ฉาก | พูด/ทำ |
|---|---|---|
| 0:00-0:20 | คลิปทางตัดวัดใจ (จาก dataset) | "ทุกวันนี้คนไทยฝ่าเครื่องกั้นรถไฟกันจนเป็นเรื่องปกติ... นี่คือ RailGuard AI" |
| 0:20-0:50 | หน้าแดชบอร์ด | ชี้ KPI: เหตุฝ่าฝืนที่ตรวจพบ, สถิติอุบัติเหตุประเทศ, กราฟตามประเภท/จุด |
| 0:50-1:40 | หน้าวิดีโอ | เปิด overlay ของอโศก/รามคำแหง: อธิบาย zone แดง = เขตราง, เส้นขาว = เส้นหยุด, สถานะ ACTIVE ขณะรถไฟผ่าน, กรอบแดง = ผู้ฝ่าฝืน |
| 1:40-2:40 | หน้าเหตุฝ่าฝืน | คลิกเหตุการณ์ → snapshot + คลิปหลักฐาน → กด "สร้างรายงาน AI" สด ๆ → อ่านรายงาน: สรุปเหตุการณ์ + มาตรา 62/63 + ปรับ 1,000 บาท |
| 2:40-3:10 | หน้าถามกฎหมาย | พิมพ์ "ฝ่าเครื่องกั้นทางรถไฟ โทษเท่าไหร่" → คำตอบพร้อม citation chips |
| 3:10-3:40 | ปิดท้าย | สถาปัตยกรรมย่อ 1 ภาพ + eval ตัวเลข + vision: ต่อ CCTV จริง 600+ ทางตัดทั่วไทย |

Tips: อัดหน้าจอ 1080p, พูดไทย, ใส่ zoom-in ตอน AI จับผู้ฝ่าฝืน, จบด้วยโลโก้ SAIG challenge
